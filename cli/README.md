# @vaultmesh/cli

VaultMesh 命令列同步工具（佔位）。

## 概述

提供終端機介面操作 VaultMesh，預計支援：

```
vaultmesh init        初始化知識庫並連線至伺服器
vaultmesh pull        從伺服器拉取最新變更
vaultmesh push        推送本地變更至伺服器
vaultmesh status      顯示同步狀態與待同步檔案
vaultmesh config      管理連線設定
vaultmesh invite      邀請協作者加入 vault
```
